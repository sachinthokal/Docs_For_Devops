# Linux Installation

Installing Linux is simple. There are multiple ways to install it depending on your needs.

## Ways to Install Linux

* **Dual Boot**: Installing Linux alongside your existing operating system (like Windows). You can choose which OS to open when you start your computer.
* **Virtual Machine (VM)**: Running Linux inside a software (like VirtualBox or VMware) without removing your current operating system.
* **Live USB**: Running Linux directly from a USB drive to try it out without making permanent changes to your computer.

## Basic Steps for Installation

1. **Download an ISO file**: Choose a Linux distribution (like Ubuntu, Fedora, or Mint) and download its ISO file from the official website.
2. **Create a Bootable USB**: Use tools like Rufus or BalenaEtcher to flash the ISO file onto a USB drive.
3. **Boot from USB**: Restart your computer, enter the BIOS menu, and select your USB drive to boot.
4. **Follow the On-Screen Instructions**: Click "Install", choose your language, format your disk partition, create a username and password, and complete the installation.
