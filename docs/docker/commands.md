# Linux Commands

Linux relies heavily on command-line tools. These commands help you manage files, directories, and system settings quickly.

## Essential Linux Commands

| Command | Description | Example |
| :--- | :--- | :--- |
| `ls` | Lists files and folders in the current directory | `ls -l` |
| `cd` | Changes the current directory | `cd Documents` |
| `pwd` | Prints the path of the current working directory | `pwd` |
| `mkdir` | Creates a new directory (folder) | `mkdir projects` |
| `rm` | Deletes files or directories | `rm file.txt` |
| `cp` | Copies files or directories | `cp file.txt backup/` |
| `mv` | Moves or renames files or directories | `mv old.txt new.txt` |
| `sudo` | Runs a command with superuser (administrator) privileges | `sudo apt update` |

## Tips

* Use the **Tab** key to auto-complete your commands or file names.
* Type `clear` to clean up your terminal screen.
